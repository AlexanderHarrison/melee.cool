New Fire Transformation Clips!

Wavedash Clip

We can clip into stadium using a well timed wavedash here. This is well known and commonly done on accident.

Clip: Move into the corner. Jump and f1 airdodge down right.

If our character don't have a tether, this clip is bad. Using an airdodge to clip causes us to enter special fall and fall to our death. We die with no way to recover or have fun inside the stage. We want to be actionable after clipping.

Airdodge Item Throw?

We can cancel the first 3 frames of airdodge with an item throw. This is called an airdodge item throw (or an aerial glide toss, but I think that's a bad name). Can we use that here to become actionable after the wavedash clip?

Clip: Move into the corner. Jump and f1 airdodge down right. Item throw in the first 3 frames of airdodge.

No. An airdodge item throw retains our velocity from before the airdodge. The jump velocity makes us pop up from beneath the stage.

This could work if we could stop our vertical momentum before the airdodge. And Peach has just the tool! Her double jump!

DJ Airdodge Item Throw Clip

Clip: Move into the corner. Jump and f1 double jump, then f1 airdodge down right. Item throw in the first 3 frames of airdodge.

This clip works but it's very hard. It requires a frame perfect double jump, a frame perfect airdodge, and a frame perfect left stick to airdodge down and right without double jumping to the right. And then hit the 3 frame airdodge item throw.

If we play around with this clip a bit, we find that we don't need the airdodge at all! This might be an easier clip.

DJ Item Throw Clip

Clip: Move into the corner. Jump and f1 double jump, then next frame hold right and item throw.

Unfortunately, this clip is still really hard! We need a frame perfect left stick to move right after the double jump. And we also need a frame perfect cstick throw input after the double jump. We will clip back up if we are off. The airdodge item throw clip might have had more inputs, but fast double frame perfect analog inputs are hard and inconsistent.

DJ Aerial Clip

There's nothing special about an item throw here. We are using it to cancel our upwards double jump velocity, but we can also cancel double jump velocity using aerials.

Clip: Move into the corner. Jump and f1 double jump, then next frame hold right and input an aerial.

This setup isn't any easier, but it doesn't need a turnip. For a long time I thought this was the easiest clip. But we can do way better!

Aside: This method of clipping was popularized (discovered?) by PracticalTas. He used an inconsistent tas-only setup. I quickly improved his setup into this one and hit it legit. That's how I started researching clip setups.

The New Setup

If you practice the old hellish setup enough, you will eventually (by accident) do this:

Which is really really strange, because usually you double jump land like this:

Somehow, we are clipping into the stage by just double jumping? Since we spend more time below the floor with this wacky double jump, it could lead to easier setups. How can we replicate this?

If we look into it, we find that the double jump clip occurs when Peach is very slightly further to the left than usual. Here, the double jump clip works at -49.582, but not at -49.831.

Getting this far left from walking is kinda random. We need to walk fast for a single frame without entering dash. The faster we walk, the further to the left we end up. If we walk fast enough, the double jump clip works.

We can play around with every method to moving into the corner and see if they consistently move us far enough left. The options are walking, dashing, running, wavedashing, dj landing, dash attacking, glide tossing, dash jc throwing, dash jc grabbing, dash jc usmash, landing turnip throws, landing aerials, and dash turnip throw.

Only two work. The walk setup we used before, and dash turnip throw!

Hooray! We found a consistent setup to get this wacky double jump clip. We just need to turnip dash throw into the corner! Because this wacky double jump causes us to be inside the stage for a good while, we should be able to clip more consistently. How much more consistent are our old clip methods with this new strategy?

Aerial Clip with New Setup

Let's try the double jump aerial clip first:

Clip: Turnip dash throw into the corner. Jump and f1 double jump, then hold right and uair at various timings. Here, we hold right f2 and uair f6.

Yes! The double jump aerial clip is more lenient with this new setup. However, it's still really inconsistent. The timing of when this clip works is very complicated. Different aerials work with different timings and it depends on when and how you drift. And polling on the left stick can cause you to randomly fail or succeed.

Airdodge Item Throw Clip with New Setup

Likewise, we can also try the airdodge item throw. Because we have a large window to airdodge with the new setup, it's MUCH more lenient then the old setup.

Clip: Turnip dash throw into the corner. Jump and f1 double jump. Within 9 frames, airdodge down right and item throw in the next 3 frames.

Note that dthrow is slightly more lenient than other throws. This setup is very consistent. The only frame perfect input is the double jump. However, it is an awkward input because you need to release jump before the airdodge to avoid floating, and there are a lot of buttons to press and sticks to move. But when you get the timing down it should be pretty consistent.

The best part of this clip method? Sometimes when you fail, you still clip! If you mistime the airdodge press, you still throw the turnip and can get the dj throw clip. And if you don't release jump before holding down right, you get a new type of clip - the dj float clip!

DJ Float Clip

Clip: Turnip dash throw into the corner. Jump and f1 double jump. Hold down right to float, then quickly release jump to drop float.

The dj float clip is by far the easiest and most consistent method to clip. However, it doesn't leave us with many options. We need to use float and double jump to clip, so we can't do much while inside the stage. It's impractical to grab the right ledge using this clip. TLDR

TLDR: You can clip by walking into the corner, instant double jumping, then either item throwing, using an aerial, or airdodge item throwing. These clips are really really hard and inconsistent. By turnip dash throwing into the corner, you move slightly further left than usual. From this position, our instant double jump will clip much more than usual. Then, we can use the same setups from before and they will be way more lenient. We can now also dj float clip, which is extremely lenient but gives us few options for fun after clipping.

How Does Clipping Work?

So how do these clips work?

Collision in a stage is made up of connected lines. White lines are floors, blue lines are left-facing walls, green lines are right-facing walls, and red lines are ceilings. This part of stadium lets us stand on a wall. We can sometimes go downwards into a wall because it expects us to move into it from the side. From here, we can move beneath the floor and clip into the stage to the right. To be honest, I'm not sure about the specifics, I just try stuff until it works.

Right Side Clip?

The above clips work because the wall is at an angle, not straight up and down. This other part of the fire transformation has this property, can we clip here too? All we should need to do is move downwards into the wall, then move right while beneath the stage.

I won't bore you with all the setups I tried. I tried a billion different methods to clip, and, turns out, we can clip with a simple airdodge! I don't know why this hasn't been found before.

Clip: Move up to wall. Shorthop and drift into the wall. Airdodge down right the frame before landing.

Right Side Clip Airdodge Item Throw

And when we airdodge, we can airdodge item throw! I love airdodge item throw. Because we airdodge when we are falling instead of rising, our retained velocity moves us down into the stage instead of back up in-bounds.

Clip: Move up to wall. Shorthop and drift into the wall. Airdodge down right the frame before landing, then item throw in the next 3 frames.

I really have no clue why this hasn't been discovered before. Maybe this is known, but I haven't seen anything on it.

YL and Link's Right Side Clip

Link and Young Link can do this clip too, using bombs! They need to shorthop from the little bottom dip instead of next to the wall though.

Clip: Move into bottom dip. Shorthop and drift into the wall. Airdodge down right the frame before landing, then item throw in the next 3 frames.

Nice and simple! Once you get the timing down this clip is pretty consistent. Make sure that you press down right as late as possible. It's around a 2 frame window.

I'm sure there are more setups to clip here, but this is the only one I could find. If I understood how these clips worked better, I'm sure I could find more.

Left Side Falling Clip?

Remember that we can only airdodge item throw clip when we are falling. Otherwise we pop up from beneath when our pre-airdodge velocity is restored. On the left side, we solved this by airdodging after a double jump. On the right side, we solved this by airdodging at the bottom of a short hop. But we never checked if we could clip by airdodging after a short hop or another falling method on the left side!

Unfortunately, there's nowhere to shorthop from on the left side. If you shorthop and try to fall into the wall, you just land on the platform. But there's another method to fall - from float!

Left Side Float Airdodge Clip

Again, I tried a billion setups here. But the setup that works is the simplest. Run up to the wall, ground float, drop float, and airdodge item throw down right on the last falling frame.

Clip: Pull a turnip. Run into the wall and ground float. Drop float and hold down right. Airdodge on the last frame before landing, then item throw in the next three frames.

This clip has A LOT going for it. It's definitely the best left side clip if you don't need float. You can throw just by pressing A and you will throw backwards, turning around to face the hidden inner ledge. This avoids a tight analog cstick input. Sometimes, the airdodge has a 2 frame window. Not sure when exactly. After clipping you have a very large window to double jump to grab the hidden inner ledge.

There is one tricky part about this clip: having the right x velocity. After airdodge item throwing, your velocity before the airdodge is retained. If you are moving too fast to the left, you will clip in-bounds below the tree. If you are moving too fast to the right, the airdodge won't clip into the stage. I find the best method is to ground float with momentum to the left from dash jumping, then start holding down right just before releasing float. This seems to consistently give the correct x velocity.

Parasol Dash from Hidden Ledge

Bonus: Peach has a parasol dash from the hidden inner ledge. You can grab this ledge using parasol after clipping. The spacing is tricky because you can't see anything, but you should be able to get it down. After grabbing this ledge from parasol, quickly release ledge, double jump, then airdodge straight up. It's slightly more lenient than a regular parasol dash.

TLDR

And that's every method I know to clip into stadium by yourself after the transformation has settled! There are 8 methods from 4 different setups in total:

  1. Left side, no setup:
    • wavedash (easy, consistent, dead)
    • instant dj aerial/item throw (hard, inconsistent, fun)
    • instant dj airdodge item throw (hard, inconsistent, fun)
  2. Left side, turnip dash throw setup:
    • instant dj aerial/item throw (hard, inconsistent, fun)
    • instant dj airdodge item throw (hard, consistent, fun)
    • instant dj float (easy, consistent, meh)
  3. Left side, dash ground float setup:
    • drop float, airdodge item throw (easy, consistent, fun)
  4. Right side, shorthop setup:
    • airdodge item throw (easy, consistent, fun)

What Should You Learn?

I suggest you start by practicing the ground float airdodge clip first. Just run and ground float into the corner, then fall and airdodge down right item throw. It's easy and consistent when you get the timing down!

If you want float after clipping, learn the right-side airdodge item throw clip. This one is also pretty easy and very simple, but can be inconsistent.

Enjoy!
